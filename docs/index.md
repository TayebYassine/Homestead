## Features

- [x] Lightweight, free, and open-source project.
- [x] Customizable and language file editable.
- [x] Advanced flags, permissions, and configuration.
- [x] Supports more than 7 integrations.
- [x] Economy, including renting and selling lands.
- [x] Trust and ban system for players.
- [x] Supports more than 5 different databases + Migration supported.
- [x] Dynamic web rendering maps.
