## Reporting an Issue

Before reporting, please:

1. Check the [FAQ](../Getting Started/FAQ.md) and [Troubleshooting](Troubleshooting.md) guide
2. Make sure you're using the latest version
3. Search existing GitHub issues

When reporting, include:

- **Plugin version** (e.g., 5.2.2.2)
- **Server software & version** (e.g., Paper 1.21.10)
- **Description of the bug or exploit**
- **Steps to reproduce**
- **Any error messages from console**

!!! warning "Unsupported Versions"

    Issues with obsolete plugin versions will be automatically ignored. Always use the latest release.
