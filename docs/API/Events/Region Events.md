# Region Events

- **RegionBanPlayerEvent**: Triggers once a player gets banned from a region.
- **RegionChatEvent**: Triggers once a player sends a message in private chat of a region.
- **RegionCreateEvent**: Triggers once a player creates a new region.
- **RegionDeleteEvent**: Triggers once a player deletes a region.
- **RegionRenameEvent**: Triggers once a player updates the region's name.
- **RegionTransferOwnershipEvent**: Triggers once a player transfers ownership to another player, or a player bought a region.
- **RegionTrustPlayerEvent**: Triggers once a player trusted a player, or an invited player accepts an invitation.
- **RegionUnbanPlayer**: Triggers once a player gets unbanned from a region.
- **RegionUntrustPlayer**: Triggers once a player gets untrusted from a region, or a player left a region manually.
